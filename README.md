# exempleDjango

## Installation

### création env

```bash
py -m venv myenv
```

### passage en environnement virtuel
pas de source sur cmd.exe
```bash
source myenv\Scripts\activate 
```

### téléchargement pip 

```bash
pip install -r requierments
```

### lancement du projet 

```bash
cd exempleDjango
py manage.py runserver
```
